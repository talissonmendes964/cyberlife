# CyberLife

Pequeno app React (Vite) com uma tela inicial contendo um botão "START" e um slideshow de fundo.

Como usar

1. Coloque as imagens no diretório `public/images/` com os nomes:
   - `1.jpg`
   - `2.jpg`
   - `3.jpg`
   - `10.jpg`
2. Instale dependências:

```powershell
cd c:\Users\Crmv\Desktop\cyberlife
npm install
```

3. Rode em modo de desenvolvimento:

```powershell
npm run dev
```
